# Caishen Readable for Codex Dream Skin

This is a portable `.cds-theme.zip` source package for Codex Dream Skin
1.10.0+.

It contains only public-safe artwork and theme metadata:

- `manifest.json`
- `theme.json`
- `background.jpg`
- `README.md`
- `LICENSE.txt`

It does not include private Codex workspace screenshots, chats, task names,
project names, file paths, or user data.

Gallery:
https://codex-theme-gallery.howardhua.chatgpt.site/themes/caishen-readable

Release package:
https://github.com/ChannelerH/codex-skin-packs/releases/download/v0.1.0/caishen-readable.cds-theme.zip

Validation:

```bash
node macos/scripts/theme-package.mjs validate --archive caishen-readable.cds-theme.zip
```
