# Caishen Readable

This is a public-safe Codex Dream Skin theme pack.

## Files

- `theme.json`: theme tokens and metadata for Codex Dream Skin.
- `background.png`: the skin image referenced by `theme.json`.

## Notes

- Built as an independent Codex theme experiment.
- Not affiliated with OpenAI.
- No private Codex workspace screenshots are included.
- The theme is intentionally light and readable: text panels stay high contrast, and the illustration is kept away from the left working area where possible.
