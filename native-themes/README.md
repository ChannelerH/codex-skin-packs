# Native Codex Theme Imports

These files are for the native Codex Settings / Appearance / Import path.
They do not install a wallpaper runtime and do not patch the Codex app.

## Caishen Readable Light

- `caishen-readable-light.codex-theme.txt`: full paste-ready `codex-theme-v1:` string.
- `caishen-readable-light.codex-theme-v1`: compatibility copy of the same import string.
- `caishen-readable-light.json`: raw theme payload without the `codex-theme-v1:` prefix.

Use the full import string when Codex asks for a paste-ready theme. Use the raw
JSON only when Codex asks for theme JSON.
